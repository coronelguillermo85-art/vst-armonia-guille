# SENSITIVE VST - Armonía Completa

Asistente didáctico de detección de acordes en tiempo real. Este proyecto forma parte del ecosistema **Sensitive**, enfocado en la creación de herramientas tecnológicas para músicos y productores.

## 🚀 Características
- **Detección en tiempo real:** Análisis de armonía instantáneo.
- **Web Audio API:** Procesamiento de audio de baja latencia directamente en el navegador.
- **Tone.js:** Framework robusto para la lógica musical.

## 🛠️ Tecnologías utilizadas
- HTML5 / JavaScript
- [Tone.js](https://tonejs.github.io/)
- Web Audio API
- Vercel (Deployment)
